# NEXUS: THE GREAT TECHNOLOGICAL LEAP

Una plataforma enciclopédica interactiva sobre los avances tecnológicos del siglo XXI, diseñada con una estética vanguardista y una IA funcional integrada.

## Características
- **16 Módulos Educativos:** Explicaciones profundas sobre IA, CRISPR, Computación Cuántica, y más.
- **Nexus Brain IA:** Chat interactivo con búsqueda semántica local.
- **Diseño de Descubrimiento:** Interfaz inmersiva que revela contenido mediante interacción.
- **Estética Híbrida:** Minimalismo tecnológico combinado con ilustraciones estilo Snoopy (Peanuts).

## Publicación
Este proyecto está diseñado para ser publicado en GitHub Pages.
